# 🚗 Cars CRUD System (.NET 8)

Sistema de gestión de vehículos construido con ASP.NET Core MVC + Web
API.

## Tecnologías

-   ASP.NET Core MVC
-   Web API
-   HttpClient
-   C#

## Arquitectura

Cars.Web (MVC) Cars.API (REST) Cars.Domain (Entities)
Cars.Infrastructure (Data)

## Funcionalidades

-   Create Car
-   Edit Car
-   Delete Car
-   List Cars

## Configuración

BaseUrl en appsettings.json: { "ApiSettings": { "BaseUrl":
"http://localhost:5119/" } }
